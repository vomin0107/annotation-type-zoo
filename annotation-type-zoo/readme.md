coco - JSON annotations used with EfficientDet and Detectron 2.
createML - Proprietary JSON format used with Apple's CreateML and Turi Create tools.
voc - Common XML format for local data munging (commonly use for segmentation)
yoloDarknet - TXT annotations used with YOLO Darknet (both v3 and v4) and YOLOv3 PyTorch.
yolov4 - TXT annotations used with YOLOv4(deprecated).
yolov5 - TXT annotations and YAML config used with YOLOv5.
TFOD - CSV format used with Tensorflow (usually converted before training so you probably want to export as a TFRecord instead unless you need to inspect the human-readable CSV).
retinanet - A custom CSV format used by the Keras implementation of RetinaNet.
tfrecord - Binary format used for both Tensorflow 1.5 and Tensorflow 2.0 Object Detection models.
